# 画面遷移ドキュメント

## 1. 画面一覧

| ID | 画面名 | ルート | 認証 |
|---|---|---|---|
| S1 | ランディング / ログイン | `/` | 不要 |
| S2 | メール送信完了 | `/auth/check-email` | 不要 |
| S3 | マイアルバム一覧 | `/albums` | 必要 |
| S4 | アルバム新規作成（モーダル） | `/albums?new=1` | 必要 |
| S5 | 画像アップロード | `/albums/[id]/upload` | 必要 |
| S6 | 解析中（進捗表示） | `/albums/[id]/analyzing` | 必要 |
| S7 | 推薦結果（7枚） | `/albums/[id]/recommend` | 必要 |
| S8 | 選別調整（追加/除外） | `/albums/[id]/select` | 必要 |
| S9 | スタイル選択 + 自由記述 | `/albums/[id]/style` | 必要 |
| S10 | 加工中 | `/albums/[id]/transforming` | 必要 |
| S11 | 結果プレビュー / DL | `/albums/[id]/result` | 必要 |
| S12 | シール注文（サイズ・枚数） | `/albums/[id]/order` | 必要 |
| S13 | 注文確認 → SUZURI 遷移 | `/albums/[id]/order/confirm` | 必要 |
| S14 | 注文履歴 | `/orders` | 必要 |
| L1 | 利用規約 | `/legal/terms` | 不要 |
| L2 | プライバシーポリシー | `/legal/privacy` | 不要 |
| L3 | 特定商取引法表記 | `/legal/tokushoho` | 不要 |

## 2. 遷移フロー図

```mermaid
flowchart TD
    S1[S1: ランディング / ログイン] -->|メール入力| S2[S2: メール送信完了]
    S2 -->|Magic Link クリック| S3[S3: マイアルバム一覧]

    S3 -->|新規作成| S4[S4: アルバム作成モーダル]
    S4 -->|作成| S5[S5: 画像アップロード]
    S3 -->|既存を開く| S5

    S5 -->|アップロード完了| S6[S6: 解析中]
    S6 -->|解析完了 自動遷移| S7[S7: 推薦結果 7枚]

    S7 -->|そのまま進む| S9[S9: スタイル選択 + 自由記述]
    S7 -->|追加/除外したい| S8[S8: 選別調整]
    S8 -->|確定| S9

    S9 -->|加工実行| S10[S10: 加工中]
    S10 -->|完了 自動遷移| S11[S11: 結果プレビュー / DL]
    S10 -->|失敗| S11_err[エラー表示・再生成]
    S11_err --> S9

    S11 -->|ダウンロードのみ| END1[完了]
    S11 -->|シールにする| S12[S12: シール注文]

    S12 -->|サイズ・枚数確定| S13[S13: 注文確認]
    S13 -->|SUZURI へ| EXT[外部: SUZURI 商品ページ]
    EXT -->|戻る| S14[S14: 注文履歴]

    S3 -.->|履歴を見る| S14
```
